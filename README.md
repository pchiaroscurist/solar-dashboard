# Solar Project Dashboard

## Live Demo
[URL](https://solar-dashboard-k7bwsgr5yferbkrrppeze4.streamlit.app/)

## Setup Instructions
### Prerequisites
- Python 3.8+
- Git
### Setup
- Download and open the repository
- Install dependencies listed in requirements.txt
- Run the app in terminal (python -m streamlit run app.py)

Your browser will open to the locally hosted version of the app

## File Structure
- app.py (Streamlit frontend)
- model.py (Financial model backend)
- requirements.txt (Python package dependencies)
- README.md (This file!)
